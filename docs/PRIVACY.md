# 隐私政策（占位模板 · 上线前需法务确认）

> 本文件为 MVP 阶段占位模板，用于满足 Gate 1「隐私政策上线」的最低要求。
> **正式上线前必须由法务/合规审阅并替换为符合《个人信息保护法》《数据安全法》《网络安全法》的正式文本。**

## 1. 我们收集的信息

- 账号信息：邮箱、昵称、加密后的密码（scrypt 加盐哈希，不存明文）。
- 学习数据：行测作答记录、错题本、模拟面试会话与报告。
- 技术信息：为防滥用而临时使用的 IP（仅用于限流，不长期留存）。

## 2. 信息用途

- 提供并同步备考进度、错题本、面试记录等核心功能。
- 保障账号与接口安全（鉴权、限流、异常追踪）。

## 3. AI 生成内容

- 模拟面试的提问、追问与评分报告由大语言模型生成，**仅供备考参考**，见 [AI 内容免责声明](./AI-DISCLAIMER.md)。

## 4. 数据存储与安全

- 密码经 scrypt 加盐哈希存储；会话通过 HttpOnly + 签名 Cookie 承载。
- 生产环境数据存于 PostgreSQL；传输使用 HTTPS。

## 5. 第三方共享

- 除法律要求外，不向第三方出售个人信息。
- 使用的第三方服务（LLM/搜索/错误追踪）仅在提供功能所必需范围内处理数据。
