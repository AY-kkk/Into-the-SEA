# 用户协议（占位模板 · 上线前需法务确认）

> 本文件为 MVP 阶段占位模板，满足 Gate 1「用户协议上线」的最低要求，正式上线前须由法务审阅替换。

## 1. 服务内容

本产品提供公务员/国企备考相关的招录情报、岗位备考、行测刷题、申论案例与模拟面试等功能。

## 2. 账号

- 你须提供真实邮箱注册，并对账号下的活动负责。
- 禁止利用自动化手段滥用接口（已部署限流与配额）。

## 3. 内容与知识产权

- 题库、申论案例等内容标注来源（sourceUrl），部分来自公开/开源数据，其著作权归原权利人所有。
- 引用/改编内容仅用于学习目的；商用前须二次确认授权（见 [数据来源](./DATA-SOURCES.md)）。

## 4. AI 生成内容免责

详见 [AI 内容免责声明](./AI-DISCLAIMER.md)。招录信息以官方公告为准。

## 5. 服务变更与终止

我们可能因维护、合规或不可抗力调整或中止服务。

## 6. 责任限制

在法律允许范围内，本产品对因使用 AI 生成内容或第三方数据造成的损失不承担责任。
